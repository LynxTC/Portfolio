關於我
About Me

我是一名對吸收事物充滿動力的學習者。過往以來，除了接收本科的資訊及商業資訊外，我也不斷活用校園提供的資源，探索各種領域的知識。因為在這個快速變化的科技時代，保有人文素養與知能才是做為人的獨特之處以及競爭力所在。但程式技能的鞏固仍是相當重要的一部分，因此我保持著持續學習的熱情，不斷探索新的工具與方法。

專業技能
Professional Skills

在大學的學習過程中，我努力累積各方面的技術能力。從前端介面到後端系統，我都有嘗試開發的經驗，也很喜歡與同學一起組隊協作。我正在學習如何朝現代化的開發流程前進，並練習寫出更好維護、效能更好的程式碼。

除了寫程式，我也在課堂與專案中學習使用各種工具來增加效率，像是 Git 版本控制或雲端部署等。省下來的更多時間是為了下一個專案而留，不斷的精進與站在使用者的角度思考，才能為這個世界提出有用的方案，以及帶來改變的力量。

對我來說，保持學習的熱情很重要。我會持續挑戰自己，並把學到的新技術應用在未來的專案中。

NCCU Pro－政大個人化學程潛能分析師

這是一個開源工具，旨在解決政大學生無法快速得知修課歷程與學程要求的差距，以及無法有效找到與興趣專長相符的學程之問題。透過上傳從校務全人系統匯出的「課業學習」JSON 檔案，即可快速檢核學分學程或微學程的修習進度，並提供智慧推薦功能。
本專案使用 Go 與 Vue 3 開發，支援 PWA 與資料不落地隱私保護。
